## Architecture

```
src
├── App.tsx: root component
├── components: Common components
├── constants: constants
├── hooks: hooks
├── lib: libraries including api client
├── main.ts: entry point
├── navigation: navigation
├── screens: screens
├── store
│   ├── auth: auth store
│   │   ├── actions.ts
│   │   ├── reducer.ts
│   │   └── saga.ts
│   ├── index.ts: create store
│   ├── reducer.ts: reducer root
│   └── saga.ts: saga root
└── styles: style files
```
